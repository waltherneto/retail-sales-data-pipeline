# Retail Sales Data Pipeline

An end-to-end batch data engineering portfolio project that ingests raw retail sales data from CSV, cleans and transforms it with Python and Pandas, loads it into PostgreSQL, models analytics-ready tables in a star schema, and serves business insights to Power BI.

## Tech Stack
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Project Structure
- `data/`: raw and processed datasets
- `etl/`: extraction, transformation, and loading logic
- `sql/`: DDL, transformations, analytics, and data quality checks
- `analysis/`: business questions and Power BI assets
- `docs/`: architecture and schema diagrams

## Current Status
Bootstrap completed. ETL foundation and project structure are in place.
